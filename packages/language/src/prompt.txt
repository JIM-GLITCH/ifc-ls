示例：

TYPE IfcAbsorbedDoseMeasure = REAL;
END_TYPE;
翻译为
IfcAbsorbedDoseMeasure :
    content = REAL;

TYPE IfcArcIndex = LIST [3:3] OF IfcPositiveInteger;
END_TYPE;
翻译为
IfcArcIndex :
    "(" content += IfcPositiveInteger "," content += IfcPositiveInteger "," content += IfcPositiveInteger  "," ")"
其中[3:3]表示列表元素最少3个 最多3个
